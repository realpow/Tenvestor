# PRD: 종합 투자 인텔리전스 플랫폼

**버전**: v1.0  
**작성일**: 2026-06-18  
**상태**: 초안

---

## 1. 프로젝트 개요

### 1.1 목적
기존 SKT 뉴스 브리핑 사이트를 확장하여, 국내외 주요 투자 정보를 한 곳에서 확인할 수 있는 종합 투자 인텔리전스 플랫폼을 구축한다.

### 1.2 핵심 가치
- 매일 아침 투자 판단에 필요한 정보를 한 화면에서 확인
- 수동 검색 없이 조건에 맞는 종목을 자동으로 필터링
- 기존 환율·DART 기능을 그대로 유지하며 기능 확장

### 1.3 타겟 사용자
국내 주식 및 글로벌 빅테크 투자에 관심 있는 개인 투자자

| 탭 2. 조건 검색 | 코스피/코스닥 종목 기술적·재무적 조건 스크리닝 |

## 3. 탭 1: 실시간 뉴스

### 3.1 섹션 구성

#### A. 미국 증시 (최상단 고정)
- 간밤의 미국 3대 지수 결과: 다우존스, S&P500, 나스닥
- 표시 항목: 지수명 / 종가 / 전일 대비 등락(%, pt)
- 데이터 소스: 무료 API 활용 (확인 필요: Yahoo Finance API 등)

#### B. 글로벌 빅테크 뉴스 (MANGOS + 기존 기업)
대상 기업:
- **MANGOS**: Meta, Apple, Nvidia, Google, OpenAI, Samsung(삼성전자)
- **추가**: Microsoft, Amazon, Tesla, TSMC
- 기존 사이트의 SKT·KT·LG유플러스는 아래 통신 섹션으로 이동

#### C. 국내 산업별 뉴스

### 3.2 기존 기능 유지
- 환율 티커 (최상단 고정, 한국수출입은행)
- 뉴스 키워드 티커 (NOW 바)
- 검색 / 공유 / 자동 갱신 (5분)
- Signal Rail (hot/warm/cool 색상 인코딩)
- DART 기업개황 조회 (서브탭)

## 4. 탭 2: 조건 검색

### 4.1 검색 모수
- **대상**: 코스피 + 코스닥 전 종목
- **필터**: 시가총액 2,000억 원 이상
- **데이터 소스**: pykrx, yfinance, DART (확인 필요: 서버사이드 실행)

### 4.2 조건 목록 (7가지)

#### 조건 1. 장기 바닥 횡보
- 일봉 기준 최소 3개월 ~ 2년간 박스권 횡보
- 기준: 52주 고가 대비 현재가 -20% 이내, 변동성(ATR) 낮음
- 찰리 멍거 철학: 오랫동안 방치된 저평가 우량주 발굴
- 출력: 횡보 기간(월), 박스권 범위(%)

#### 조건 2. RSI 과매도
- 일봉 RSI ≤ 20
- 주봉 RSI ≤ 25
- 두 조건 동시 충족 종목만 출력
- 출력: 일봉 RSI, 주봉 RSI, 현재가

#### 조건 3. 거래량 폭증 + 주가 급등
- 당일 거래량이 전일 대비 10배 이상
- 당일 주가 상승률 5% 이상
- 출력: 거래량 배율, 상승률(%), 현재가

#### 조건 4. 실적 고성장
- 최근 3개년 연간 매출 성장률 30% 이상
- 최근 3개년 연간 영업이익 성장률 30% 이상
- 두 조건 동시 충족
- 데이터 소스: DART 재무제표
- 출력: 3개년 매출 추이, 3개년 영업이익 추이, 성장률(%)

#### 조건 5. 볼린저밴드 상단 돌파
- 일봉 기준 현재가 ≥ 볼린저밴드 상단(20일, 2σ)
- 출력: 현재가, BB 상단값, 이격률(%)

#### 조건 6. 엔벨로프 하단 접촉
- 일봉 기준 현재가 ≤ 엔벨로프 하단(20일 이동평균 -10%)
- 출력: 현재가, 엔벨로프 하단값, 이격률(%)

#### 조건 7. 이동평균선 수렴
- 일봉 + 주봉 기준
- 5일선·10일선·20일선·60일선이 ±3% 이내로 수렴
- 출력: 각 이평선 값, 최대 이격률(%)

### 4.3 UI 구성
- 조건 7개를 카드형으로 나열
- 각 카드: 조건명 / 설명 / [검색 실행] 버튼
- 검색 결과: 종목명, 종목코드, 현재가, 시가총액, 조건별 핵심 수치
- 결과 클릭 시: 해당 종목 DART 기업개황 또는 외부 차트로 연결
- 전체 결과 CSV 다운로드 버튼

## 5. 기술 요구사항

### 5.1 프론트엔드
- 기존 바닐라 JS + HTML + CSS 스택 유지
- 탭 전환: 기존 mode-nav 패턴 확장 (뉴스 탭 / 조건검색 탭)
- 새 CSS 변수 추가 없이 기존 디자인 시스템 100% 재사용

### 5.2 백엔드 (신규 API)
```
GET /api/us-market          # 미국 3대 지수
GET /api/stock-screen       # 조건 검색 실행
  ?condition=rsi_oversold   # 조건 코드
  &min_cap=200000000000     # 시총 필터 (2천억)
GET /api/stock-detail       # 종목 상세 (차트 데이터)
```

### 5.3 데이터 파이프라인
- 주가/재무 데이터: pykrx + yfinance (기존 GitHub Actions 파이프라인 활용 가능)
- 뉴스: Google 뉴스 RSS (기존 방식 유지, 대상 기업·키워드만 변경)
- 미국 지수: Yahoo Finance API 또는 Alpha Vantage (확인 필요)
- 재무 데이터: DART OpenAPI (기존 연동 확장)
